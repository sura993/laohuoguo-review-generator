# 火锅店AI好评生成系统

## 项目概述
这是一个为火锅店设计的智能化好评生成系统。顾客在就餐后，通过触碰NFC卡片可以直接跳转到小红书APP，系统会自动生成一篇200字以内的高质量好评文案，顾客确认后即可手动发布。

## 功能特性
1. **NFC卡片跳转** - 触碰NFC卡片直接跳转到好评生成页面
2. **AI智能生成** - 基于就餐体验自动生成高质量好评文案
3. **小红书集成** - 一键跳转小红书APP发布界面
4. **自定义编辑** - 顾客可自由编辑生成的文案
5. **多语言支持** - 支持中文好评生成

## 技术架构
- **前端**: HTML5 + CSS3 + JavaScript (兼容移动设备)
- **AI引擎**: 本地AI文本生成或云端API
- **NFC技术**: NDEF协议配置跳转链接
- **部署**: 静态网页部署，支持HTTPS

## 使用流程
1. 顾客就餐完毕后触碰NFC卡片
2. 手机自动打开网页或跳转小红书
3. AI生成个性化好评文案
4. 顾客编辑确认文案
5. 点击"发布到小红书"按钮
6. 跳转小红书APP发布页面

## 部署要求
- HTTPS域名（NFC跳转需要安全连接）
- NFC卡片编程设备
- 智能手机支持NFC功能
